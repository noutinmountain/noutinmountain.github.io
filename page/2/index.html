解决连接github中443问题
发表于2022-01-04|更新于2022-07-17|工具使用|github| 条评论
问题：今天打开vscode试图克隆github上的项目但是报443连接失败，前提：我是全天24小时挂着代理，网上搜索的解决办法太垃圾了（关掉代理）自己尝试解决了 一，查看ip点开第一个连接查看github的IP地址：https://github.com.ipaddress.com/ 将网页中的 IP Address 复制下来 后面备用 点击第二个连接：https://github.com.ipaddress.com/assets-cdn.github.com 将网页中的 IP Address 复制下来 后面备用 点击第三个连接：https://github.com.ipaddress.com/assets-cdn.github.com 同理记住IP address 二，配置host使用命令： sudo nano /private/etc/hosts 输入上一步查看的ip地址 三，刷新DNSsudo killall -HUP mDNSResponder;say DNS cache has been flushed 再尝试ping github成功，完美解决
